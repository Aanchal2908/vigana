# Vigana

React + Vite based frontend app.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages deployment

Project is configured for auto-deploy with GitHub Actions.

1. Push code to `main` branch.
2. In GitHub repo, go to **Settings > Pages**.
3. Set **Source** to **GitHub Actions**.
4. Wait for workflow **Deploy to GitHub Pages** to pass.
5. Live URL will be available at:
   `https://aanchal2908.github.io/vigana/`
